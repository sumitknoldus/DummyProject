Introduction This is a small application which gives brief detail of periodic elements in tabular format. Technologies Used This application has been built upon following technologies:
  • Angular15+
  • Angular Material.
  • NodeJS
  • Typescript
  • HTML, CSS
 Layout This application consist two sections currently
  • Header Navbar
  • Element Table
 Features (What it Does?) This application provides below few key features:
  • Drag Drop feature on table columns so that users can arrange them as per their convenience.
  • Pagination and Sorting is applied on the data for better readability.
 Testing Guidelines This application provides below few key features:
  • Verify that user should see "Evolving Web" brand image on the header section.
  • Verify that user should see burger icon button as well as settings icon on the header nav bar.
  • Verify that Sorting and Pagination is working accordingly.
  • Verify that user is able to see the data in the EW Table.
